# 青岛旅游宣传网站

这是一个基于 HTML/CSS/JavaScript 开发的青岛旅游宣传网站，用于展示青岛的景点、美食、旅游路线等信息，是我的 Web 前端课程实践项目。

## 技术栈
- HTML5
- CSS3
- JavaScript
- Swiper.js（轮播图插件）

## 项目结构
qingdao-tour-website/
├── css/ # 样式文件
├── images/ # 图片资源
├── js/ # JavaScript 脚本
├── index.html # 首页
├── about.html # 关于我们
├── food.html # 特色美食
├── route.html # 旅游路线
└── README.md # 项目说明

## 运行方式
直接双击 `index.html` 文件，在浏览器中打开即可查看完整网站效果。

## 项目亮点
- 采用模块化布局，代码结构清晰易维护
- 使用 Swiper.js 实现轮播图，提升页面交互体验
- 包含景点介绍、美食推荐、旅游路线等多个功能模块
- 后续可扩展响应式布局，适配电脑端设备
